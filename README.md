# ASAD Creative Studio

ASAD Creative Studio is a modern digital portfolio website created by Asad Arisar. The website showcases creative work including web design, app design, AI image creation, and AI video production.

## Features

* Modern and responsive portfolio website
* User login and signup system
* Admin dashboard for managing messages
* Contact form connected with Supabase database
* Creative portfolio showcase
* Clean and professional UI design

## Technologies Used

* HTML
* CSS
* JavaScript
* Supabase (Authentication and Database)

## Website Sections

* Home
* About
* Portfolio
* Services
* Contact
* Login / Signup
* Admin Dashboard

## Purpose

The purpose of this website is to showcase creative digital work and provide professional design services to businesses, startups, and individuals.
